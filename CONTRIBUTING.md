# 🤝 Contributing to BuddyAI Desktop

Thank you for your interest in contributing to **BuddyAI Desktop** — a desktop-native AI workspace built for real-world productivity.

We welcome contributions from developers, architects, designers, and AI enthusiasts.

---

## 🚀 Ways to Contribute

You can contribute in several ways:

- 🐛 Bug fixes
- ✨ New features
- 🧠 AI provider integrations (OpenAI, Claude, Grok, Ollama, etc.)
- 🎨 UI/UX improvements (WinForms + WebView2)
- 📄 Documentation improvements
- 🔌 Persona library contributions
- ⚙️ Performance optimizations

---

## 📋 Before You Start

1. Check existing **Issues** to avoid duplicates  
2. Open a new issue if:
   - You are proposing a new feature
   - You are planning a large change  
3. Keep changes **focused and minimal**

---

## 🛠️ Development Setup

### Prerequisites

- Visual Studio 2022
- .NET 8 SDK
- Windows OS (WinForms dependency)
- WebView2 Runtime installed
